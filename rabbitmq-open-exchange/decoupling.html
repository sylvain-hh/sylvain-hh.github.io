The new Open Exchange introduce some funny questions about general pubsub mechanisms. The main idea is simple : in main protocols, fo pubsub to work, Producers and Consumers MUST share something common : the "topic" pattern.
This implies that the so called "decoupling system" is not so "decoupled", because if this pattern changes, we must change Producer AND Consumer.
